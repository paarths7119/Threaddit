
# Threaddit

MERN Reddit-clone starter using Tailwind, JWT auth, and MongoDB Atlas.

## Setup

### Backend
1. cd backend
2. copy `.env.example` to `.env` and fill values
3. npm install
4. npm run dev

### Frontend
1. cd frontend
2. npm install
3. copy `.env.example` to `.env` and (optionally) adjust `VITE_API_URL`
4. npm run dev

Open the frontend URL shown by Vite (usually http://localhost:5173). The backend runs on port 5000 by default.
